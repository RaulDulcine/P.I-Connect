Descrição do Projeto

Sistema web completo para gestão de atividades escolares, incluindo controle de alunos, professores, notas, frequência e comunicação entre escola e família. O projeto visa democratizar o acesso à tecnologia educacional e melhorar a qualidade do ensino através da digitalização de processos.

Objetivos

  • Facilitar a gestão acadêmica de instituições de ensino
  • Melhorar a comunicação entre escola, alunos e responsáveis
  • Automatizar processos administrativos e reduzir uso de papel
  • Fornecer dados para tomada de decisão pedagógica

Tecnologias Utilizadas

React, Node.js, PostgreSQL, TypeScript, Tailwind CSS
